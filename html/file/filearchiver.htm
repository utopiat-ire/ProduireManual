概要

各種書庫ファイルを扱う手順を集めた種類です。

手順

解凍する　圧縮する　書庫ファイルとして開く　

次の手順が使用できます。

【書庫ファイル名】を【解凍先】へ解凍する

書庫ファイルを解凍します。
ZIP形式以外の書庫ファイルの解凍には各種統合アーカイバDLLが必要です。
【オプション】は、圧縮方法をDLL専用のパラメータ文字列で指定します。 

「書庫.lzh」をデスクトップへ解凍する

保護あり　保護機能の有効時は、解凍先にシステム領域を指定できません。

【ファイル・フォルダ】を【書庫ファイル名】へ〈【オプション】で〉〈【基準フォルダ】から〉圧縮する

ファイルを圧縮します。
ZIP形式以外の書庫ファイルへの圧縮には各種統合アーカイバDLLが必要です。
【オプション】は、圧縮方法をDLL専用のパラメータ文字列で指定します。【基準フォルダ】を指定すると、書庫にそのフォルダを基準に格納します。

対象フォルダを「［デスクトップ］書庫ファイル.zip」へ圧縮する

保護あり　保護機能の有効時は、書庫ファイル名にシステム領域を指定できません。

【ファイル名】を書庫ファイルとして開く：書庫ファイル

指定したファイルを書庫として扱います。

「書庫ファイル.zip」を書庫ファイルとして開いて、書庫とする
書庫をデスクトップへ解凍する

設定項目

設定項目 型 　 説明
統合アーカイバ使用 真偽値 ◎ 統合アーカイバプロジェクト準拠のDLLを優先して使用するかどうか(既定値は〇)
×とするとZIP形式のみ標準機能が利用されます。

記号の意味

サンプルコード

zipファイルなどを解凍する

書庫ファイルを解凍するには「解凍」手順を使います。

書式

【書庫ファイル】を【解凍先のフォルダ】へ解凍する

【書庫ファイル】には、各種アーカイバで圧縮された書庫ファイルを指定します。
【解凍先のフォルダ】には、書庫ファイルの内容を展開する先のフォルダを指定します。

書庫ファイルの拡張子に応じて、使用するアーカイバを自動的に選択します。

例文　archive.zipをデスクトップのarcフォルダへ解凍します

「archive.zip」を「［デスクトップ］arc\」へ解凍する

zipファイルを作成する

ファイルやフォルダを圧縮するには「圧縮」手順を使います。

書式

【圧縮するファイル・フォルダ】を【書庫ファイル】へ圧縮する

【圧縮するファイル・フォルダ】には、圧縮するファイルまたはフォルダを指定します。
【書庫ファイル】には、圧縮先の書庫ファイルを指定します。

書庫ファイルの拡張子に応じて、圧縮に使用するアーカイバを自動的に選択します。

例文　マイドキュメントにある「文章」フォルダを「archive.zip」へ圧縮します

「［マイドキュメント］文章\」を「archive.zip」へ圧縮する
