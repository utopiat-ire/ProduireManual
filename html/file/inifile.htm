利用プラグイン

ファイル操作機能

概要

INIファイルを操作する種類です。

作成する方法

【名前】という設定ファイルを作る

手順

次の手順が使用できます。

読み込む　取得する　書き込む　設定する　削除する　セッション名を列挙する　見出しを列挙する　すべて取得する　

【自分】について【セクション名】から【キー名】を〈【既定値】で〉読み込む：文字列

設定ファイルの指定した【セクション名】に含まれる【キー名】に保存されている値を読み込みます。
指定したセッション名またはキー名が存在しない場合は、〈既定値〉が返されます。

【自分】について【セクション名】から【キー名】を〈【既定値】で〉取得する：文字列

「書き込む」手順と同じです。

【自分】について【セクション名】で【キー名】へ【値】を書き込む

設定ファイルの指定した【セクション名】に含まれる【キー名】に【値】を格納にします。

【自分】について【セクション名】で【キー名】へ【値】を設定する

「書き込む」手順と同じです。

【自分】について【セクション名】から〈【キー名】を〉削除する

設定ファイルから指定した【セッション名】または【キー名】を削除します。
【キー名】を省略すると、セクション全体が削除されます。

【自分】についてセッション名を列挙する：文字列の配列

設定ファイルのすべてのセクション名を列挙します。

【自分】について【セクション名】から見出しを列挙する：文字列の配列

設定ファイルの指定した【セッション名】のすべての見出しを列挙します。

【自分】について【セクション名】からすべて取得する：辞書

設定ファイルの指定した【セッション名】のすべての見出しと値を辞書として取得します。

設定項目

次の設定項目が使用できます。

設定項目 型 　 説明
ファイル名  ◎ 操作する設定ファイル(INIファイル)

サンプルコード

INIファイルを作成したり取得したりする

設定ファイルを作成して値を取得します。

ーー初期設定
アプリ設定という設定ファイルを作る
アプリ設定のファイル名は「[デスクトップ]settings.ini」

ーー項目を設定します
アプリ設定について「全般」で「利用者名」へ「高橋」を設定する

ーー項目を取得します
利用者名は、アプリ設定について「全般」から「利用者名」を取得したもの
利用者名を表示する

ーー項目を削除します
アプリ設定について「全般」から「利用者名」を削除する

.urlファイルからアドレスを取得する

Internet ExplorerのURLショートカットからURLを取得します。

URLショートカットという設定ファイルを作る
URLショートカットのファイル名は「プロデル公式サイト.url」
URLショートカットについて「InternetShortcut」で「URL」へ「https://rdr.utopiat.net/」を設定する

Bという設定ファイルを作る
Bのファイル名は「プロデル公式サイト.url」
URLは、B：「InternetShortcut」から「URL」を取得したもの
URLを表示する
