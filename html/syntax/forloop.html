繰り返し文

  • 指定した回数だけ繰り返す
  • 変数が指定した値になるまで繰り返す
  • 条件に応じて繰り返す
  • 無制限に繰り返す
  • 配列の内容に応じてそれぞれ繰り返す
  • 繰り返しの動きを変える
  • ブロック式で指定する

繰り返し文

同じプログラムを繰り返して実行するには「繰り返し」文を使います。

  • プロデルで始める日本語プログラミング言語入門(#4) 「繰り返す」

繰り返し文には、いくつかの書き方があります。

指定した回数だけ繰り返す

指定した回数だけ、プログラムを繰り返すには、次のように書きます。

書式

《繰り返す回数》回、繰り返す
　　/* 繰り返したい文 */
そして

《繰り返し回数》で、繰り返し回数を書きます。

※「そして」は、旧文法の「繰り返し終わり」,「繰り返し終」とも書けます

例文　3回「わん」を表示する

3回、繰り返す
　　「わん」を表示する
そして

繰り返した回数をカウントする

繰り返した回数を変数にカウントするには、次のように書きます。

書式

《繰り返す回数》回《変数名》にカウントしながら繰り返す
　　/* 繰り返したい文 */
そして

《変数名》にカウントする変数を書きます。

※「そして」は、旧文法の「繰り返し終わり」,「繰り返し終」とも書けます

例文　3回、カウントする

3回、値にカウントしながら繰り返す
　　「[値]回目です。」を表示する
そして

変数が指定した値になるまで繰り返す

指定した範囲で、処理を反復するには、次のように書きます。

書式

《変数名》を《初期値》から《増減数》ずつ増やしながら《最終値》まで繰り返す
　　/* 繰り返したい文 */
そして
  • 《変数名》に、カウントする変数を書きます。
  • 《初期値》に、変数の初期値を書きます。(省略可能)
  • 《増減数》に、一回実行するごとに加算する数値(マイナスを含む)を書きます。省略すると1と同じです。
  • 《最終値》に、変数の最終値を書きます。(省略可能)

※「そして」は、旧文法の「繰り返し終わり」,「繰り返し終」とも書けます

例文

値を10から2ずつ増やしながら20まで繰り返す
　　値を表示する
そして

条件に応じて繰り返す

特定の条件に応じて繰り返せます。

書式　条件を満たしている間は繰り返す

《条件式》の間、繰り返す
　　/* 繰り返したい文 */
そして

《条件式》が成立している(○である)間、繰り返します。

※「そして」は、旧文法の「繰り返し終わり」,「繰り返し終」とも書けます

例文　１から５まで表示します

値は、１
値が５以下の間、繰り返す
　　値を表示する
　　値は、値＋１
そして

条件式を満たすまで繰り返す

書式

《条件式》になるまで繰り返す
　　/* 繰り返したい文 */
そして

《条件式》が成立する(○になる)まで繰り返します。

※「そして」は、旧文法の「繰り返し終わり」,「繰り返し終」と書くこともできます

例文　１から５まで表示します

値は、０
（値が５）になるまで繰り返す
　　値は、値＋１
　　値を表示する
そして

無制限に繰り返す

繰り返し文で、繰り返し条件を省略すると、無制限に繰り返します。繰り返しから抜け出す場合は「繰り返しから抜け出す」文を使います。

無限に繰り返すには、次のように書きます。

書式

繰り返す
　　/* 繰り返したい文 */
そして

※「そして」は、旧文法の「繰り返し終わり」,「繰り返し終」とも書けます

例文　[いいえ]をクリックするまで繰り返します

繰り返す
 　　「続けますか？」を質問して結果とする
 　　結果が×なら、繰り返しから抜け出す
そして

配列の内容に応じてそれぞれ繰り返す

配列の内容に対して、繰り返して処理する場合は「それぞれ繰り返し」文を使います。配列については「配列」のページをご覧ください。

書式

《配列》を《要素変数》へそれぞれ繰り返す
　　/* 繰り返したい文 */
そして

《配列》
繰り返す内容が含まれる配列を書きます。

《要素変数》
それぞれの配列の要素を格納するための変数を書きます。

※「そして」は、旧文法の「繰り返し終わり」,「繰り返し終」とも書けます

ひ

例文　配列の内容を順番に表示します

一覧は｛「N-03B」,「F-04B」,「P-09A」,「SH-08B」｝

一覧を要素へそれぞれ繰り返す
　　要素を表示する
そして

日本語らしい書き方

それぞれ繰り返し文は、次のようにも書けます。

名前一覧は｛「裕太」,「健司」,「智美」,「直子」｝

名前一覧のすべての名前について『
　　名前を表示する
』ことをそれぞれ繰り返す

 

繰り返しの動きを変える

繰り返し文の動作を変えられます。

繰り返しを抜け出す

繰り返し回数に関係なく、繰り返しから抜け出すには「繰り返しを抜け出す」文を使います。

3回、変数にカウントしながら繰り返す
　　「わん」を表示する
　　繰り返しを抜け出す
そして

繰り返しを続ける

繰り返し文の処理を飛ばして、処理をはじめに戻すには「繰り返しを続ける」文を使います。

3回、繰り返す
　　「わん」を表示する

　　繰り返しを続ける
　　//この次からは実行されません
　　「にゃん」を表示する
そして

ブロック式の使用

ブロック式を使って、次のようにも書けます。

書式

《繰り返す回数》回、『
 		/* 繰り返したい文 */
』ことを繰り返す

例文　3回「わん」を表示する

3回『
　　「わん」を表示する
』ことを繰り返す
