利用プラグイン

データ機能

概要

「JSON形式」は、JSON形式を操作するための部品です。

手順

読み取る　書き出す　整形して書き出す　

【内容:文字列】をJSON形式として読み取る:辞書

内容に指定されたJSON文字列を辞書へ変換します。

【内容:辞書】をJSON形式として書き出す

内容に指定された辞書をJSON文字列へ変換します。

【内容:辞書】をJSON形式として整形して書き出す

内容に指定された辞書を整形されたJSON文字列へ変換します。

設定項目

設定項目 型 　 説明
インデント文字列 真偽値 ◎ インデントに用いるスペース文字列

記号の意味

サンプルコード

JSON形式を文字列として生成する

JSON形式を作って、JSON形式と辞書とを相互変換します。

値は{「プロデル」,{バージョン=「1.2.500」}}
JSON形式として値を書き出したものを報告する

JSON形式の文字列から辞書を作る

//単純
値は、JSON形式:「{
"userID":"ajax",
"userPW":"xanadu" , 
"name" : "\u4eca\u671d\u306f",
"count":1.05
}
」を読み取る
値を報告する

JSON形式の文字列から配列と辞書を作る

//複雑
値は、JSON形式:「{"item":[[]
{"itemCode":91,
"itemName":"塩ラーメン",
"itemPrice":300},

{"itemCode":94,
"itemName":"味噌ラーメン",
"itemPrice":290},

{"itemCode":95,
"itemName":"豚骨ラーメン",
"itemPrice":320}
[]]
}
」を読み取る
値を報告する
