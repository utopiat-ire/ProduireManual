静的種類保護あり

  • 概要
  • 手順
  • 設定項目

利用プラグイン

システム情報取得

概要

Windowsに対する操作や情報を取得するための種類です。

手順

次の手順が使用できます。

スタンバイする　ログオフする　休止状態する　再起動する　終了する　

【自分】をスタンバイする

コンピュータをスタンバイ状態にします。

Windows XPまで対応しています。Windows Vista以降には対応していません。

ウィンドウズをスタンバイする

【自分】をログオフする

ログオフします

ウィンドウズをログオフする

【自分】を休止状態する

コンピュータを休止状態にします。

Windows XPまで対応しています。Windows Vista以降には対応していません。

ウィンドウズを休止状態する

【自分】を再起動する

コンピュータを再起動します

ウィンドウズを再起動する

【自分】を終了する

ウィンドウズを終了します

ウィンドウズを終了する

設定項目

次の設定項目が使用できます。

設定項目 型 　 説明
コンピュータ名 文字列 □ 起動しているコンピュータ名を表します
バージョン 文字列 □ Windowsのバージョンを表します
ビルド番号 整数 □ Windowsのビルド番号を表します
ビット数 整数 □ Windowsの動作ビット数を表します
ユーザ名 文字列 □ 現在ログインしているユーザ名を表します
種類 文字列 □ Windowsの種類(通称名)を表します
名称 文字列 □ Windowsの版名を含む詳細な名称を表します
起動中プログラム一覧 文字列配列 □ 現在起動しているプログラムの一覧を表します
