リストビューのイベント情報

イベント情報は、イベント手順が実行された時に、その手順内で有効な「この時」変数で取得できます。

記号の意味

ラベル編集イベント情報

設定項目 種類 　 説明
キャンセル 真偽値 ◎ 編集後の内容
番号 整数 □ 編集している項目の番号
編集後 文字列 □ 編集後の内容

見出しがクリックされたイベント情報

設定項目 種類 　 説明
見出し番号 整数 ◎ 選択している見出しの番号

見出しの幅が変更された時イベント情報

設定項目 種類 　 説明
見出し番号 整数 ◎ 選択している見出しの番号

仮想項目が必要になった時イベント情報

設定項目 種類 　 説明
見出し番号 整数 ◎ 選択している見出しの番号

仮想項目が必要になった時イベント情報

設定項目 種類 　 説明
項目番号 整数 □ 選択している見出しの番号
項目   ◎ 実際に表示する項目
